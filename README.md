# LBC (Layout Block Component) - Builder
------------
This is a WebPack static sites builder.

Author: Vitaliy Shebela (www.linkedin.com/in/vitaliy-shebela)

### Requirements
------------

* Node.js (>= 6.6.0)
* NPM (>= 3.10.3)

### Installation
------------
```sh
$ npm install
```
